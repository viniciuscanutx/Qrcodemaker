#QRCODEMAKER




##O que o QrCode Maker faz?


###O qrcode maker é capaz de fazer um qrcode com qualquer tipo de link ou frase que você colocar nele, inclusive cardápios de restaurantes e etc.


##O QrCode Maker é feito em que linguagem?


###O QrCode Maker é feito totalmente em python usando algumas bibliotecas do próprio python.



#O Aplicativo pode conter alguns bugs, mas funciona corretamente.

##Os arquivos ficam salvos em uma pasta chamada QrCode que será criada no mesmo diretório que o .exe estiver.
